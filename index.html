GARIGORI 프로젝트를 소개 합니다.

GARIGORI 프로젝트는 한국센차유저그룹(KoreaSenchaUserGroup)에서 진행하는 오픈소스 프로젝트로 현재 대학생들의 직업선택에 도움을 주고 있는 Garigori 앱을 만드는 모임 입니다. Sencha Touch 기술을 사용하고, iOS/Android 를 타겟하고 있습니다.

한국 센차 유저 그룹

  • 커뮤니티 : Sencha 기술을 연구하는 모임으로써 발표 혹은 기고 등을 통해 연구 자료를 공유하는 모임 입니다.

  • 한국센차유저그룹은 Sencha 기술을 다루는 커뮤니티 입니다. Sencha Touch/Ext JS/Sencha Architect 등을 다루며 공유하고 있습니다. 연구 자료는 발표/연제/전자책 형태로 공유하고 있으며 1년에 2번 대외발표(별천지 세미나)를 하고 있습니다. (블로그:http://sencha.or.kr , 페이스북 : https://www.facebook.com/groups/korea.sencha)

  • URL

    • Facebook : https://www.facebook.com/groups/korea.sencha

  • 현황

    • 커뮤니티 : 2012년도 설립, 회원수 576명(페이스북 활동 기준)
    • 주요 활동
      • Sencha 전자책 발간 (센차 기술 및 연구자료, 번역, 세미나 자료) ( URL : http://sencha.or.kr/?p=755 )
      • 기술지(월간 마이크로소프트웨어) 연재 활동 ( URL : http://sencha.or.kr/?p=771 )
      • 연구 자료의 Github 공개 ( URL: https://github.com/KoreaSenchaUserGroup )
      • 다수의 정기 세미나 발표 및 스터디 활동
      • 사회적 기업을 위한 Application 제작 프로젝트 Sencha 기술을 이용하면 Application을 빠르게 만들 수 있는 장점이 있습니다. 사회에 좋은 기여를 많이 하지만, 앱이 없는 기업이나 단체를 위해 재능 기부로 무료 제작을 해주고 있으며, 제작 과정을 통해 스터디를 병행하고 있습니다. (현재 대학생들의 직업선택에 도움을 주고 있는 Garigori 프로젝트 진행 중 : http://garigori.sencha.or.kr )

  • 진행중인 프로젝트

    • ESTAR 프로젝트 : 자폐 장애학생의 사회/경제 참여를 지원하는 사회적 기업(autistar.co.kr)을 위한 앱 개발 (2013년 1월 ~ 현재 진행 중)
    • GARIGORI 프로젝트 : 현재 대학생들의 직업선택에 도움을 주고 있는 Garigori 프로젝트 진행 중
